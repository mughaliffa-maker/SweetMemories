# The Sweet Crumbs CrumbCam

## Open in VS Code
1. Extract this ZIP.
2. Open the extracted folder in VS Code.
3. Right click `index.html` and choose **Open with Live Server**.

## Upload to Vercel
Upload/import this exact folder, not the parent folder.
The project has `package.json` and `vercel.json`, so Vercel will build the static site into `dist` automatically.

## Important
Camera access works on localhost or HTTPS only.
